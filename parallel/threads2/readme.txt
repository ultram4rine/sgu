Напишите программу, которая  будет подсчитывать количество цифр «2», используемых в десятичной записи целых чисел от 0 до n (включительно). Иметь возможно задать число потоков и интервал десятичных чисел для каждого потока. Когда будет происходит прибавление к счетчику должен выводить номер потока и число на текущий момент подсчитанных двоек.
